# Photos de l'équipe

Placez ici les photos des membres de l'équipe. Les fichiers sont servis à la racine du site.

**Noms de fichiers utilisés dans TeamSection :**
- `ceo.jpg` — photo du CEO (Romel Matsonda)
- `marie.jpg` — Marie Laurent (CTO)
- `pierre.jpg` — Pierre Dubois (Cybersécurité)
- `sophie.jpg` — Sophie Martin (UI/UX)

**Recommandations :**
- Format : JPG ou WebP
- Taille : au moins 400×400 px (carré ou recadrable)
- Poids : idéalement < 200 Ko par image

Vous pouvez utiliser d'autres noms : il suffit de modifier les `avatarUrl` dans `app/components/TeamSection.tsx`.
